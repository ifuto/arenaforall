Features

低スペック機のために、
ぜんぶ作り直した。

「重い」の原因を一つずつ特定し、実測で裏付けながら潰していく。それが Rsift のやり方。

🧊

12バイト頂点フォーマット

位置 u16×3 + 八面体法線 + uv の量子化。Sodium の 20B より小さく、帯域を 40% 削減。

VERTEX_STRIDE = 12
🗜️

メッシュコンパクター

1万セクションの全カリング＋圧縮が 111.8µs。1 フレームに 149 回まわせる。

実測 8,901 ops/s
🧵

スタッターガード

フレームアリーナで 4096 alloc + reset が 6.2µs。一時確保の GC 振動をほぼゼロへ。

実測 151,492 ops/s
🏗️

Nanite 風クラスタリング

最大 128 三角形のメッシュレット化＋終了保証つき flood-fill。背面クラスタを事前除去。

culling 前段で捨てる
🎛️

クオリティガバナ

重い区間を検知して自動降段 → 軽くなったら復帰。設定を触らず安定 FPS。

実測 683,761 ops/s
💾

Zstd リージョン I/O

64KiB チャンクの圧縮＋展開が 101.9µs。ボトルネックのディスク待ちを置き去りに。

実測 9,756 ops/s
